# Keep Theme Starter


## Get start | 快速开始

### Install dependencies | 安装依赖

```bash
npm install
```

### Run the project | 运行项目

```bash
npm run server
```

### Configuring theme | 配置主题

Please configure the `source/_data/keep.yml` file.

请对 `source/_data/keep.yml` 文件进行配置。
